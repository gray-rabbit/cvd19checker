## 사용법 

설치를 하시고..

필수 정보를 채워주세요.

### 필수 정보 입력

1. 선생님의 성함, 코로나 자가진단 코드, 학년, 반을 입력해 주세요 (필수정보입니다.)

<img width="500" src="https://user-images.githubusercontent.com/9082094/83776640-5f8b0000-a6c3-11ea-8f09-3cb9fc964551.png"/>



### (선택사항 - 선택사항을 채우시면 소통알리미를 통해 문자를 발송할 수 있습니다.)

1. 소통알리미의 아이디를 확인해 주세요.

2. 소통알리미의 패스워드를 확인해 주세요

3. 우리반 그룹명을 넣어 주세요.

4. (중요) 소통알리미에 등록된 선생님의 휴대폰 번호를 넣어주세요(숫자만 넣어주세요).

5. 나이스에서 코로나 관련 엑셀파일을 다운 받으신 후, 학생 부분의 전체 데이터를 선택 후 복사(Ctrl+C) 붙여넣기 (Ctrl+v)를 이용해 붙여 넣습니다. 

6. 데이터 저장하기를 눌러주세요.

<img width="500" src="https://user-images.githubusercontent.com/9082094/83777023-e17b2900-a6c3-11ea-8e30-32b85d4d8dce.png"/>

#### 기타사항(소통알리미 우리반 그룹명 찾기)

먼저 소통알리미로 접속해 주세요. 소통메신저 상단의 소통알리미를 이용하시면 됩니다. 

1. 소통알리미의 아이디와 패스워드를 확인해 주세요.

<img width="500" src="https://user-images.githubusercontent.com/9082094/83776355-09b65800-a6c3-11ea-97fe-5a1339feba83.png"/>

2. 소통알리미에서 *신규발송*를 눌러 주세요.

<img width="500" src="https://user-images.githubusercontent.com/9082094/83778531-cd382b80-a6c5-11ea-90cf-64be5be603e5.png"/>

3. 주소록에서 자신의 학급의 문자를 확인합니다. (제 경우 학년별 1학급 뿐이므로 '2학년'이 됩니다.)

<img width="300" src="https://user-images.githubusercontent.com/9082094/83778797-243e0080-a6c6-11ea-9bd1-279e2f6dcb77.png"/>




### 자동 갱신 및 문자 보내기

1. 메인을 클릭하고 자동갱신 할 간격을 입력하고 갱신하기를 클릭합니다.(갱신의 단위는 분입니다.)

<img width="500" src="https://user-images.githubusercontent.com/9082094/83779152-8dbe0f00-a6c6-11ea-8481-95d4581e0ee4.png"/>


2. 갱신을 하고 나면 결과가 나타납니다. 분단위로 자동 갱신되며 갱신을 해제할 수 있습니다. 

3. 미제출자(결과없음으로 나타나는 것이 미제출자 입니다.)에게 한번에 모두에게 문자를 보낼 수 있습니다.(이때 보내지는 문자는 소통알리미에서 확인이 가능하며, 각각의 학생에게 개별 링크를 전송하게 됩니다.)

4. 확인됌! 버튼은 선생님께서 직접 확인처리를 하는 기능입니다. 학생이 학교에 왔으며, 구슬로 문진을 한 후에 사용하세요(절대로 악용하지 마세요. 문제 발생시 모든 것은 선생님의 책임이 됩니다. 

5. 재촉하기 버튼은 개별학생에게 문자를 전송하게 됩니다.
